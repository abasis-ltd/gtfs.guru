Developer First

Integrate validation directly into your pipelines with our robust CLI and Python bindings.

Python API

import gtfs_validator

result = gtfs_validator.validate("feed.zip")

if result.is_valid:
    print(f"Success in {result.time}s")
else:
    print(f"{result.error_count} errors found")

Rust CLI

cargo run --release -- \
  --input ./gtfs.zip \
  --output ./report \
  --json
